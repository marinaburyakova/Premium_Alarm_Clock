##  **О проекте**

Premium Alarm Clock - это современное веб-приложение для управления временем с элегантным дизайном и богатым функционалом. Приложение сочетает в себе красивый интерфейс в стиле Bento Grid с эффектами неоморфизма и стекломорфизма, плавные GSAP анимации и полный набор инструментов для работы со временем.

###  **Дизайн концепция**
- **Bento Grid** - модульная сетка с органичными карточками
- **Neo-морфизм** - мягкие тени и объемные элементы
- **Glassmorphism** - эффекты стекла и размытия
- **Премиум визуал** - дорогие градиенты и тени

---

##  **Особенности**

###  **Будильник**
- Создание неограниченного количества будильников
- Настройка повторения по дням недели 
- Включение/выключение одним переключателем
- Звуковое оповещение с Web Audio API
- Браузерные уведомления
- Авто-отключение через 30 секунд

### ⏱ **Таймер**
- Предустановленные интервалы (5, 10, 15, 25, 30, 45, 60 минут)
- Визуальный прогресс-бар с анимацией
- Звуковой сигнал по окончании
- Кнопки управления: Старт, Пауза, Сброс
- Плавные GSAP анимации

###  **Dormir - Звуки для сна**
- 4 типа звуков:
- Регулировка громкости
- Бесконечное воспроизведение (loop)

###  **Часы**
- **Аналоговый циферблат:**
  - Часовая стрелка
  - Минутная стрелка
  - Секундная стрелка (красная для видимости)
  - Часовые и минутные метки
- **Цифровые часы:**
  - Отображение текущего времени
  - Обновление в реальном времени

###  **Темы оформления**
- Светлая тема (Light Theme)
- Темная тема (Dark Theme)
- Плавное переключение с анимацией
- Сохранение выбранной темы в localStorage

###  **Сохранение данных**
- Все будильники сохраняются в localStorage
- Данные сохраняются между сессиями
- Автоматическая загрузка при запуске

###  **Анимации (GSAP)**
- Stagger эффект при загрузке блоков
- Hover эффекты с увеличением карточек
- Микро-анимации при кликах
- Плавное движение часовых стрелок
- Анимации при добавлении/удалении будильников

###  **Адаптивность**
- Полная поддержка мобильных устройств
- Адаптивная верстка под все разрешения
- Удобное управление на сенсорных экранах
- Оптимизированные размеры для маленьких экранов
